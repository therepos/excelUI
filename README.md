# excelUI Add-in

## Usage

1. Download the latest [release](https://github.com/therepos/excelUI/releases/excelUI/latest/download/excelUI.zip). 
2. Install with excelUIsetup.cmd

## Structure

```
excelUI
├── .github
│   └── workflows
├── .scraps
├── app                 # Executables
├── docs                # Documentation
├── scripts             
│   └── docgen-bas.py   # Autogenerate doc/bas.md
├── src
│   └── bas             # Documentation
├── tmp                 # Temporary files
```
